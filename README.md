## What is easy Execute?
Easy execute is a system that helps executing scripts via little words!
### - Example 1:
Instead of
```luau
loadstring(game:HttpGet("https://raw.githubusercontent.com/EdgeIY/infiniteyield/master/source"))()
```
You can do:
```luau
execute("iy")
```
